# Tutorial Projects

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).


## what Did I Learn

-Creating Vue App
-Click Events
-Using v-for,v-if,v-model
-Attribute Binding
-Dynamic Classes
-Computed Properties
-Using VUE-CLI
-Template Refs
-Emiting Custom Events
-Click Event Modifiers
-Slots
-Using Teleport
-Lifecycle Hooks
-Password Validation
-Vue Router
-Dynamic Links
-Redirect & 404 
-Fetching Data
-Using Json Server
-Options API
-The Composition API
